# customer-churn-analysis
Python customer churn analysis project exploring customer retention, contract behavior, and churn trends using Pandas and Matplotlib.
## Project Overview
This project analyzes customer churn behavior using Python and Pandas. The analysis explores customer retention trends, contract types, and churn distribution.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Dataset exploration
- Data cleaning
- Churn distribution analysis
- Contract type churn analysis
- Data visualization
- Business insight generation

## Key Findings
- Month-to-month contracts have the highest churn rate
- Long-term contracts show stronger customer retention
- Customer contract type strongly impacts churn behavior
